# Local AI Inspector

> Understand your local AI workstation before trying to optimize it.

Local AI Inspector is an open-source project that helps users inspect, understand and troubleshoot local AI workstations using observed system information instead of assumptions.

The project is designed for people who want to build or maintain a reliable local AI environment without requiring deep expertise in Linux, ROCm or LLM infrastructure.

## Current focus

The initial scope intentionally remains small.

Supported platform:

- Linux
- Fedora
- AMD CPUs and GPUs
- ROCm
- Ollama

Additional platforms and runtimes may be supported in the future, but they are not part of the initial objectives.

## Principles

- Observe before analyzing.
- Recommendations must be based on collected facts.
- Never modify the system without explicit user approval.
- Keep the project simple, transparent and maintainable.
- AI should improve explanations, not replace system inspection.

## Project status

🚧 Early design phase.

The first milestone focuses on defining a solid architecture before writing production code.
